## Project Objective
To design an app prototype that helps employees manage 
workplace stress through personalized Smart Stress Relief 
Pod sessions using AI-driven recommendations.

---

## Design Thinking Process

### Empathize & Define
- Conducted surveys with 10 participants
- Created 10 personas representing different employee types
- Built empathy maps capturing stress triggers
- Problem Statement: "How might we design a system that 
offers instant, personalized relaxation for employees, 
reducing stress without disrupting workflow?"

### Ideate
Key features brainstormed and included:
1. MoodSync AI – personalized pod settings
2. Breathe with Me – guided micro-meditation
3. Pod Scheduler – session booking system
4. Post-Session Journal – stress reflection tracker
5. Stress Tracking Dashboard – progress over time
